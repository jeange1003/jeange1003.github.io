一款VSCode Code Review插件的设计

功能介绍

  • 一款VSCode插件
  • 评论当前查看的代码，接收别人的评论，并与之讨论
  • 查看，评论MR

其中，代码讨论功能包含

  • 框选一段代码，并评论。
  • 当别人@的时候，接收通知
  • 打开通知后可以看到对方评论对针对的代码，以及讨论内容
  • 点击评论中的代码，跳转至对应的代码位置
  • 评论内容包含文本与图片
  • 查看当前文件的所有讨论
  • 查看自己的所有讨论
  • 查看当前仓库的所有讨论
  • 飞书通知别人@自己的评论

Merge Request 相关功能包含

  • 查看当前仓库所有Merge Requests
  • 查看自己参与的，需要自己Review的Merge Requests
  • 查看单个 Merge Request，内容包括：标题，描述，创建者，assignee，discussion，代码diff，操作历史，不同patchset的diff
  • 对Merge Request进行评论，评论存储于GitLab
  • 代码高亮与代码跳转，包括（Goto Definition，Find All Reference等等）
